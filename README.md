## Installation

### From `npm`

```sh
npm install --save @tknf-labs/html-helper # npm
yarn add @tknf-labs/html-helper           # yarn
pnpm add @tknf-labs/html-helper           # pnpm
```

### Basic usage
```js
import { html } from "@tknf-labs/html-helper";

const htmlString = html`
  <div>
    <h1>Hello World</h1>
  </div>
`;
```
